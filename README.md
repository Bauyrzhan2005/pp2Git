# TSIS 3 Racer Game

## Run
```bash
pip install pygame
python main.py
```

## Controls
- Left / A: move left
- Right / D: move right
- Esc: return to menu during the race

## Features
- Main menu, settings, leaderboard, game over screen
- Username entry
- Traffic cars and road obstacles
- Oil, potholes, speed bumps, moving barriers, nitro strips
- Nitro, Shield, Repair power-ups
- Difficulty scaling
- Distance meter and score
- Settings saved in `settings.json`
- Top 10 leaderboard saved in `leaderboard.json`
